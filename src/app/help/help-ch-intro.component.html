1.1  What Is Morse Code Studio?

Morse Code Studio is a browser-based tool for sending and receiving Morse code (CW). It runs entirely in your web browser — there is nothing to install. It works on Chrome, Edge and other Chromium-based browsers on Windows, macOS, Linux, Android and iOS.

Core capabilities:

  • Decode — Listen to incoming Morse from a key or radio and display decoded text on screen in real time.
  • Encode — Convert typed text into perfectly timed Morse code audio and key signals.
  • Key — Send Morse manually via keyboard, mouse or touchscreen, supporting straight key and four paddle modes.
  • Key your radio — Drive your transmitter's keying line through an optocoupler (sound card channel) or a USB-serial adapter (DTR/RTS).

Everything happens in real time with sub-millisecond precision, suitable for on-air CW operating at any practical speed (5–50+ WPM).

1.2  Getting Started

Follow these steps to get up and running:

  1. Click "Start Audio" — The browser needs your permission to use the microphone and speakers. A dialog will appear the first time.
  2. Open Settings — Click the gear icon (⚙) in the header bar to open the Settings modal where you configure inputs, outputs and keyer options.
  3. Choose your inputs — Enable the input method(s) you want: Straight Key via Mic (pilot tone), Straight Key via Audio Channel (CW tone), or any combination of the keyboard, mouse or touch keyers.
  4. Choose your outputs — Enable the output(s) you need: Sidetone (speaker), Optocoupler (sound card) and/or Serial Port (DTR/RTS).
  5. Save Settings — Click "Save Settings" at the bottom. Your settings are saved per device configuration and persist across sessions.
Quick Start — Practice decoding

Click Start Audio, open the Encoder panel, type some text and press Enter. Morse audio plays through your speakers while the Decoder displays the decoded text in real time. This is a great way to practise reading Morse code by ear.
