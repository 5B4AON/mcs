Publishes decoded morse characters to a Firebase Realtime Database channel. Remote stations subscribed to the same channel and secret will receive each letter as it is decoded.

User Name is attached to each posted letter (typically your callsign). Select which decoded text to forward: RX (received), TX (transmitted by the encoder/keyer), or both.

Only the last letter per channel is stored — no message history is retained. Channel limits and automatic expiry are configured externally via Firebase Security Rules and Cloud Functions.

Requires an internet connection. Offline caching is disabled.
