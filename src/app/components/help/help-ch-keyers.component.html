4.1  Keyboard Keyer

The keyboard keyer turns your computer keyboard into a Morse key. It is always available — no hardware required.

Straight Key

A single assigned key acts as a manual Morse key. Press and hold to key down; release to key up. You control all timing by hand, just like a traditional straight key.

Paddle Modes

Two assigned keys act as dit and dah paddles. The keyer auto-generates correctly timed elements at the configured Keyer WPM. Four paddle modes are available:

ModeBehaviour
Iambic B Squeezing both paddles alternates dit-dah. Releasing completes the current alternating element. This is the most common iambic mode used by amateur radio operators.
Iambic A Same as Iambic B, but releasing both paddles stops immediately after the in-progress element finishes — no extra element is added.
Ultimatic When both paddles are held, the last pressed paddle repeats continuously. Useful for fast runs of dits or dahs.
Single Lever Each paddle repeats only its own element. No automatic alternation or memory. Works like two independent straight keys with automatic element timing.

Key Bindings

In the Keyboard Keyer section of Settings, click the button next to "Straight Key", "Left Paddle" or "Right Paddle" and press the keyboard key you want to assign. Common choices: Space for straight key, [ and ] for paddles.

Dit/Dah Memory

In Iambic modes, if you tap the opposite paddle while an element is playing, that request is remembered ("latched") and will play next. This enables faster, cleaner keying at high speeds.

Example — Iambic B keying

Assign [ = dit, ] = dah. Select Iambic B, 15 WPM. To send "C" (dah-dit-dah-dit): press ], then while the dah sounds, squeeze [, then ], then [. The keyer handles all element timing automatically.

4.2  Mouse Keyer

The mouse keyer lets you use your mouse buttons to send Morse code. This is useful when you want to key without a physical key or paddle, or as a quick alternative to the keyboard.

Enable the mouse keyer in Settings under Mouse Keyer. Each of the three mouse buttons (left, middle, right) can be independently assigned to one of four actions:

ActionBehaviour
None Button does nothing (normal browser behaviour).
Straight Key Press-and-hold keys down; release keys up. You control timing.
Dit Press activates the dit paddle. Works with the selected paddle mode (Iambic, Ultimatic, etc.) at Keyer WPM.
Dah Press activates the dah paddle. Same paddle mode behaviour.

When the mouse keyer is enabled, assigned mouse buttons are captured by the application and will not perform their normal browser actions (e.g. context menu on right-click is suppressed).

The mouse keyer shares the Paddle Mode and Keyer WPM settings with the keyboard and touch keyers.

4.3  Touch Keyer

The touch keyer provides on-screen touch buttons for sending Morse on tablets and phones. Enable it in Settings under Touch Keyer.

Two operating modes are available:

ModeBehaviour
Straight A single large touch area acts as a straight key. Touch = key down, release = key up.
Paddle The screen is split into left and right touch zones, each acting as a paddle. Works with the selected paddle mode at Keyer WPM.

Button Assignment (Paddle Mode)

In paddle mode, you can assign which element each side sends:

  • Left button — Dit or Dah (default: Dit)
  • Right button — Dit or Dah (default: Dah)

The touch keyer shares Paddle Mode and Keyer WPM with the keyboard and mouse keyers.

Haptic Vibration

Vibration output is configured in the Outputs tab of Settings, not in the Touch Keyer section. When enabled, the device vibrates while the key is down — for any input source (keyboard, mouse, touch, mic pilot, CW tone detector, or encoder), and for both TX and RX events, mirroring the sidetone output.

Android only. The Vibration API (navigator.vibrate()) is supported on Android browsers (Chrome, Firefox, Edge) but not on iOS Safari or desktop browsers. On unsupported platforms, the toggle has no effect.

4.4  Shared Keyer Settings

The following settings are shared across all three keyers (keyboard, mouse, touch):

  • Paddle Mode — Iambic B, Iambic A, Ultimatic or Single Lever. Applies to all paddle inputs regardless of source.
  • Keyer WPM — The speed at which the iambic keyer generates dits and dahs. Adjustable from the main screen via the Keyer WPM pill.

4.5  Reverse Paddles

Each keyer type has its own independent Reverse Paddles toggle in Settings. When enabled, the dit and dah paddle assignments are swapped for that keyer only.

This is useful for left-handed operators or for matching the paddle orientation of a physical key. Each keyer (keyboard, mouse, touch) can be reversed independently — reversing the keyboard paddles does not affect the mouse or touch paddles.

4.6  Straight-Key Sprite Button

A large on-screen straight-key button appears at the bottom of the main screen whenever there is enough vertical space. Tap or click it to key Morse code just like a physical straight key — press and hold for dashes, tap briefly for dots.

The sprite button is enabled by default on all devices and works independently of the touch keyer settings. It can be disabled in Settings → Other → Straight-Key Sprite.

Animation from Other Keyers

The sprite button can optionally animate (depress visually) when a straight-key press is detected from another input source. This provides a satisfying visual indicator of keying activity. In the settings card, check any combination of:

  • Keyboard straight key
  • Mouse straight key
  • MIDI straight key
  • Serial straight key
  • Straight key via mic

All animation checkboxes are unchecked by default.
