# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.2.x   | :white_check_mark: |
| < 0.2   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in Morse Code Studio, please report it responsibly.

**Do not open a public issue.** Instead, please email the maintainer directly using the contact information on the [GitHub profile](https://github.com/5B4AON), or use GitHub's [private vulnerability reporting](https://github.com/5B4AON/morse-code-studio/security/advisories/new) feature.

Please include:

- A description of the vulnerability
- Steps to reproduce it
- The potential impact
- Any suggested fix (if you have one)

You can expect an initial response within 7 days. We will work with you to understand the issue and coordinate a fix and disclosure timeline.
